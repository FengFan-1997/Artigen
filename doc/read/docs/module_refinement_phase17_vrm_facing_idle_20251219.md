# VRM 朝向修正与待机动作增强（2025-12-19）

## 本次完成

1. VRM 默认朝向更可控
   - 加载后基于骨骼（`left/rightUpperArm` + `hips/head`）推导角色“前方”向量，并在需要时自动做 `Y` 轴 180° 校正，避免部分模型出现“背对镜头/动作朝向不对”的观感问题。

2. VRM 站立基础姿态更自然（双手下垂）
   - 调整加载时的基础 Relax Pose：上臂下压与轻微内收，使默认站立时双手更接近自然下垂状态。

3. 增加长时间待机自动动作（本地即时启动）
   - 加入 VRM 端 Idle 调度：在无交互/无状态扰动/无外部动作指令时，自动轮换触发待机动作。
   - 新增/增强程序化动作：
     - `yawn`：抬手掩口哈欠后放下
     - `idle_squat_think`：下蹲托腮/思考风格待机
     - `idle_shift_weight`：站立重心轻微摇摆
     - `play_hair`：玩头发/害羞遮掩的小动作
     - `clap`：改为程序化拍手（避免播放内置 clip 时出现“拍手方向不对”）

4. 工程校验
   - `pnpm -C frontend run type-check` 通过
   - `pnpm -C frontend run lint` 通过

## 关键改动文件

- `frontend/src/agent/components/VrmWidget.vue`

   - 根据角色人设（例如“懒散/傲娇/元气”）配置不同的 idle pool 与触发频率。
